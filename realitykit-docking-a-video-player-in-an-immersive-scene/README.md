# Docking a video player in an immersive scene

## Overview

You can customize a docking region and dock a video player within any immersive scene. Create and add a fixed docking region to a video player to secure it within an immersive scene.

For more information about the app and how it works, see
[Docking a video player in an immersive scene][link-to-sample]
in the developer documentation.
