# Apple Sample Code Collection

Complete collection of Apple Developer sample code projects - extracted, cleaned, and ready to build in Xcode.

## Contents

- **606 projects** covering all Apple frameworks
- Cleaned (no .git, DerivedData, build artifacts)
- Ready to open in Xcode

## Frameworks Covered

SwiftUI, UIKit, ARKit, RealityKit, Metal, AVFoundation, CoreML, Vision, HealthKit, MapKit, CoreData, SwiftData, and 100+ more.

## Usage

```bash
# Clone the repo
git clone https://github.com/mihaelamj/cupertino-sample-code.git

# Open any project
open cupertino-sample-code/swiftui-fruta-sample/Fruta.xcodeproj
```

## Source

Downloaded from [Apple Developer Documentation](https://developer.apple.com/documentation/) using [Cupertino](https://github.com/mihaelamj/cupertino).
