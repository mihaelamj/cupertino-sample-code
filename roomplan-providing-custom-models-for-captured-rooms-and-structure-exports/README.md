# Providing custom models for captured rooms and structure exports

Enhance the look of an exported 3D model by substituting object bounding boxes with detailed 3D renditions.

For more information about the app and how it works, see [Providing custom models for captured rooms and structure exports](https://developer.apple.com/documentation/arkit/arkit_in_ios/environmental_analysis/providing_custom_models_for_captured_rooms_and_structure_exports) in the developer documentation.
