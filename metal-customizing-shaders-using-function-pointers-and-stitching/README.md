# Customizing shaders using function pointers and stitching

Define custom shader behavior at runtime by creating functions from existing ones and preferentially linking to others in a dynamic library.

## Overview

- Note: This sample code project is associated with WWDC2021 session [10229: Discover compilation workflows in Metal](https://developer.apple.com/wwdc21/10229/) and WWDC2022 session [6596: Target and optimize GPU binaries with Metal 3](https://developer.apple.com/wwdc22/6596).
