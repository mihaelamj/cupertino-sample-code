# Processing spatial video with a custom video compositor

Create a custom video compositor to edit spatial video for playback and export.

## Overview

The sample app shows how to create spatial custom video compositors, and how to use them in preview and export workflows. For a complete overview of this topic, see 
[Processing spatial video with a custom video compositor](https://developer.apple.com/documentation/avfoundation/processing-spatial-video-with-a-custom-video-compositor).
