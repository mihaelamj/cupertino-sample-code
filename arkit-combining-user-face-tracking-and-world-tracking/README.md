# Combining user face-tracking and world tracking

Track the user's face in an app that displays an AR experience with the rear camera. 

For more information about the app and how it works, see
[Combining user face-tracking and world tracking](https://developer.apple.com/documentation/arkit/arkit_in_ios/content_anchors/combining_user_face-tracking_and_world_tracking) in the developer documentation.
