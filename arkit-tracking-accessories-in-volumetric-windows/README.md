# Tracking accessories in volumetric windows

Translate the position and velocity of tracked handheld accessories to throw virtual balls at a stack of cans.

## Overview
Accessories enhance the experience of Apple Vision Pro, and offer added functionality and flexibility by supporting fine control, novel input methods, and custom experiences.
You can use ARKit to locate and track accessories, converting their real-world motion into app-accessible data.

For more information about the app and how it works, see [Tracking accessories in volumetric windows][link-to-sample] in the developer documentation.
