# Object tracking with Reality Composer Pro experiences

Use object tracking in visionOS to attach digital content to real objects to create engaging experiences.


## Overview

> Note: This sample code project is associated with WWDC24 session [100101: Explore object tracking for visionOS](https://developer.apple.com/wwdc24/10101/).

## Configure the sample code project

Simulator doesn't support ARKit, so you can only run this sample on a
physical device. This sample can run on Apple Vision Pro with 
visionOS 2 or later.
