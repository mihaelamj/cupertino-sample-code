# Adding realistic reflections to an AR experience

Use ARKit to generate environment probe textures from camera imagery and render reflective virtual objects.

For more information about the app and how it works, see
[Adding realistic reflections to an AR experience](https://developer.apple.com/documentation/arkit/arkit_in_ios/camera_lighting_and_effects/adding_realistic_reflections_to_an_ar_experience) in the developer documentation.
