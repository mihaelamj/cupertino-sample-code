# Effecting people occlusion in custom renderers

Occlude your app's virtual content where ARKit recognizes people in the camera feed by using matte generator.
   

For more information about the app and how it works, see
[Effecting people occlusion in custom renderers](https://developer.apple.com/documentation/arkit/effecting-people-occlusion-in-custom-renderers) in the developer documentation.
