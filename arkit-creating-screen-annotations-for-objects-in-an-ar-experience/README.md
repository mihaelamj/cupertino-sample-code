# Creating screen annotations for objects in an AR experience

Annotate an AR experience with virtual sticky notes that you display onscreen over real and virtual objects. 

For more information about the app and how it works, see
[Creating screen annotations for objects in an AR experience](https://developer.apple.com/documentation/arkit/arkit_in_ios/content_anchors/creating_screen_annotations_for_objects_in_an_ar_experience) in the developer documentation.
